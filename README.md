# Logistic Regression Ad Click Prediction

This project uses Machine Learning (Logistic Regression) to predict whether a user will click on an online advertisement based on demographic and internet usage data.

## Dataset Features
- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Male
- Clicked on Ad (Target Variable)

## Project Steps
1. Data Loading and Exploration
2. Exploratory Data Analysis using Seaborn
3. Train/Test Split
4. Logistic Regression Model Training
5. Predictions and Evaluation

## Libraries Used
- pandas
- numpy
- seaborn
- matplotlib
- scikit-learn

## Model Performance
Achieved approximately **91% accuracy** on test data.
